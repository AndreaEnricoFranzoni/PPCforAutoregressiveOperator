Detailed Description

Class for computing PPCKO algortihm without cross-validation: alpha as parameter, k can be a parameter or selected through explanatory power criterion.

Author
Andrea Enrico Franzoni
Note
definition of the function to make prediction on the validation set during cv process is done here: they simply wrap a PPCKO without cross-validation
