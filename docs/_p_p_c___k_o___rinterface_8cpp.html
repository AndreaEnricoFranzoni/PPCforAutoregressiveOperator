Detailed Description

Contains the R-interfaced functions of the package 'PPCKO', which implement Principal Predictive Components (PPC) Kargin-Onatski (KO) algorithm to perform one-step ahead prediction of Functional Time Series (FTS).

Author
Andrea Enrico Franzoni

Function to perform one-step ahead prediction of Functional Time Series of curves using PPCKO.

Parameters
XRcpp::NumericMatrix (matrix of double) containing the curve time series: each row (m) is the evaluation of the curve in a point of its domain, each column (n) a time instant
id_CVstring denoting which version of the algorithm the user wants to use: 'NoCV', 'CV_alpha', 'CV_k' or 'CV'
alpharegularization parameter (positive real number)
knumber of PPCs: if 0, is selected through explanatory power criterion; if between 1 and m: k is imposed
threshold_ppcminimum requested proportion of explanatory power: used only if k=0. Duble between 0 and 1
alpha_vecvector of double containing the input space for cross-validation over the regularization parameter
k_vecvector of integers containing the input space for cross-validation over the number of retained PPCs
tolltolerance over which cross-validation process stops looking for bigger number of PPCs if the validation errors between two consecutive iterations is smaller than it
disc_evvector of double containing the points of the domain for which the curve's evaluations are available
left_extremedouble indicating the left extreme of the domain of the curve
right_extremedouble indicating the right extreme of the domain of the curve
min_size_tstime instants that define the smallest training set (between 2 and max_size_ts)
max_size_tstime instants that define the biggest training set (between min_size_ts and n-1)
err_rettrue if validation errors are stored and returned, false if not
ex_solvertrue if solving PPCKO inverting the regularized covariance matrix, false if relaying on GEP to avoid id
num_threadsnumber of threads to be used in OMP parallel directives
id_rem_nanstring that defines how to handle NaNs for some instant: 'MR': replacing them with the mean of the fts in that point, 'ZR' with 0s
Returns
an R list containing:
  • one step ahead prediction of the fts
  • used regularization parameter
  • number of retained PPCs
  • scores along the PPCs (scalar product between the directions and fts at last available instant)
  • cumulative explanatory power of the PPCs (coherent only if ex_solver true)
  • directions of the PPCs
  • weights of the PPCs
  • standard deviation of the scores of each direction (evaluate on the scalar products between direction and the curve between instants 2 and n)
  • standard deviation of the scores of each weight (evaluate on the scalar products between weight and the curve between instants 1 and n-1)
  • mean function of the fts
  • validation errors (only if err_ret true)
  • points of the domain for which the evaluation of the curve are available
  • left extreme of the curve's domain
  • right extreme of the curve's domain
  • curve at the last available instant
  • which PPCKO version has been performed
  • input space for regularization parameter
  • input space for the number of PPCs
